# &lt;stl:file&gt; 属性

## no

显示字段的顺序。

## src

下载文件的地址，可以用"@"开头表示当前网站根目录，用"~"开头表示系统根目录。

## type

FileUrl 指定存储附件的字段，可以是自定义的附件字段

## isCount

是否统计此文件的下载次数。

- `"true"` 统计下载次数
- `"false"` 不统计下载次数

## isFileSize

是否统计此文件的大小。

- `"true"` 统计文件大小
- `"false"` 不统计文件大小

## leftText

显示在信息前的文字，当信息不存在时文字也不显示。

## rightText

显示在信息后的文字，当信息不存在时文字也不显示。
