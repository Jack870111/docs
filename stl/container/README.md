# &lt;stl:container&gt; 容器

```html
<stl:container
  context="所处上下文">
</stl:container>
```

## 使用说明

通过 stl:container 标签在模板中定义容器，切换上下文

`<stl:container>` 标签需要包含子标签，系统将在指定上下文情景下对子标签进行解析。

`<stl:container>` 标签对应的实体为{stl:container}。

## 属性

| 属性    | 说明       |
| ------- | ---------- |
| context | 所处上下文 |
