# &lt;stl:zoom&gt; 文字缩放

```html
<stl:zoom
  fontSize="缩放字体大小"
  isDynamic="是否动态显示"
  zoomId="页面HTML中缩放对象的ID属性">
</stl:zoom>
```

## 使用说明

通过 stl:zoom 标签在模板中实现文字缩放功能

stl:zoom 标签将显示缩放链接，点击此链接后缩放指定文字。

`<stl:zoom>` 标签对应的实体为{stl:zoom}。

## 属性

| 属性     | 说明                       |
| -------- | -------------------------- |
| zoomId   | 页面HTML中缩放对象的ID属性 |
| fontSize | 缩放字体大小               |
