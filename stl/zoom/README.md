# &lt;stl:zoom&gt; 文字缩放

```html
<stl:zoom
  zoomId="页面Html 中缩放对象的 Id 属性"
  fontSize="缩放字体大小">
</stl:zoom>
```

## 使用说明

通过 stl:zoom 标签在模板中实现文字缩放功能

stl:zoom 标签将展示缩放链接，点击此链接后缩放指定文字。

## 属性

| 属性                                    | 说明                          |
| --------------------------------------- | ----------------------------- |
| [zoomId](zoom/attributes?id=zoomId)     | 页面Html 中缩放对象的 Id 属性 |
| [fontSize](zoom/attributes?id=fontSize) | 缩放字体大小                  |
